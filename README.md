# Conexión a la API de Gemini

Este proyecto muestra cómo conectarse a la API de Gemini utilizando Python y la librería `google-generativeai`.

## Requisitos
- Python 3.10 o superior

## Dependencias
- Librerías definidas en `requirements.txt`

## Creación del entorno virtual, instalación y ejecución

```bash
python -m venv env
.\env\Scripts\activate
pip install -r .\requirements.txt
python .\app_gemini.py
#Correr ejercicio 1
python .\app_gemini_ej1.py
#Correr ejercicio 2
python .\app_gemini_ej2.py
#debes agregar primero el texto y despues la ia te preguntara que quieres hacer: Resumir, traducir etc
#Correr ejercicio 3
python .\app_gemini_ej3.py
